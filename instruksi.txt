disclaimer :

*sample app ini masih menggunakan flask dari python3
*sample app belum menggunakan stack technology (plain html & css)

instruksi windows :

	1. install docker desktop dan pastiin docker engine jalan
	2. install wsl 2 (bisa ikuti https://learn.microsoft.com/en-us/windows/wsl/install-manual#step-4---download-the-linux-kernel-update-package)
	3. buka git bash dan cd ke direktorinya (tempat sample-app.sh tersimpan)
	4. pastiin port 8080 tersedia (kalo ga tersedia matiin service yang menggunakan 8080)
	5. execute command ini di bash
		bash ./sample-app.sh
	6. buka http://localhost:8080 di browser atau curl http://localhost:8080
		buat cek udah jalan atau belum


misc :

VIA GIT BASH :

*run container
	1. execute command ini di bash
		docker container start samplerunning

*matiin container tertentu
	1. execute command ini di bash
		docker container samplerunning

*cek container docker yang berjalan :
	1. execute command ini di bash
		docker ps -a

*matiin semua container docker yang berjalan :
	1. execute command ini di bash
		docker kill $(docker ps -aq)

*delete semua container docker :
	1. execute command ini di bash
		docker rm $(docker ps -aq)
